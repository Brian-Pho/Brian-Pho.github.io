ENGG 233: Ring Runner Game

Source Code

My first major project after taking my first programming course in first year engineering. The project I chose to do was an original game programmed in Processing; a simple language based on Java. A notable problem that I had to solve when I was developing this game was trying to make the ball move in a circular path. The solution was to use sin and cos for the y and x coordinate of the ball. This made sense as the unit circle is described by trigonometric functions.
